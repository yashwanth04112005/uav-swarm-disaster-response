# Multi-Agent UAV Swarm Simulation for Disaster Response

A simulation platform for autonomous UAV swarm coordination in disaster environments.

## Features

- Disaster environment simulation
- Multi-agent UAV system
- Swarm communication
- Path planning
- Sensor fusion
- Performance analytics

## Tech Stack

- Python
- Pygame
- NumPy
- NetworkX
- Streamlit
